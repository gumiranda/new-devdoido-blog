Pipeline de YouTube

Monitore canais do YouTube, busque os vídeos mais recentes automaticamente e transcreva tudo em texto — direto no banco do seu SaaS.

<—>

Conecte uma conta Google para começar

Para consultar a YouTube Data API precisamos de uma conta Google autorizada. Não é o seu canal — essa conta serve apenas como credencial para as chamadas de leitura à API.

O que o CrazyStack faz
  • [✓] Lê metadados de vídeos públicos via API
  • [✓] Consulta os uploads mais recentes dos canais que você definir
  • [✓] Usa a quota da API em nome desta conta
O que NÃO fazemos
  • [✗] Acessar ou conectar o seu canal
  • [✗] Publicar, comentar ou curtir nada
  • [✗] Ler dados privados ou analytics da conta

Next.js 16: o que muda na prática

Rocketseat publicado 29 mai 2026 3.412 palavras capturado hoje 08:00
// transcrição completa

00:00 Fala pessoal, beleza? Hoje a gente vai falar sobre o Next.js 16 e tudo que mudou nessa versão. Tem muita coisa nova, então cola aqui que vale a pena.

00:32 A primeira grande mudança é no sistema de cache. Eles repensaram completamente como o cache funciona no App Router, e agora ficou muito mais previsível.

01:15 Antes, o comportamento padrão confundia bastante gente — você não sabia se a rota estava sendo cacheada ou não. Agora tem um controle explícito pra cada fetch.

02:08 Outra coisa importante: os Server Actions ficaram estáveis e ganharam suporte melhor pra revalidação. Vou mostrar um exemplo prático aqui no editor.

03:40 Repara que agora a gente consegue invalidar só uma tag específica do cache, sem precisar revalidar a página inteira. Isso muda o jogo pra apps grandes.

05:12 Vamos falar também do Turbopack, que finalmente saiu de beta e virou o bundler padrão pra desenvolvimento. A velocidade de hot reload melhorou demais.
