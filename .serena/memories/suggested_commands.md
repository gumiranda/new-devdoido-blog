# Suggested Commands

- `bun run dev:landing -- --host 127.0.0.1` - run Astro landing dev server.
- `bun run lint` - lint API + landing source with zero warnings.
- `bun run build:landing` - build static landing site.
- `bun run build` - build API then landing.
- `bun run test` - run API test suite.
- `rg -n "Data Center|RT-One|Uberlândia|datacenteruberlandia" apps/landing README.md SEO_GEO_REPORT.md AI_SEO_TRACKER.md third-party-presence wikipedia-articles -S` - confirm old public Data Center content is gone.
- `git status --short --untracked-files=all` - inspect worktree without destructive changes.
