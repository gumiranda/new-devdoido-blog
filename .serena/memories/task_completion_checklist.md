# Task Completion Checklist

For landing/frontend work:
1. Run `bun run lint`.
2. Run `bun run build:landing`.
3. For Devdoido migration tasks, run the old-content `rg` command and ensure no matches.
4. Start/keep a local dev server when needed and validate key routes desktop/mobile with screenshots or browser checks.
5. Check `git status --short --untracked-files=all`; report only relevant changes and leave unrelated dirty files untouched.
